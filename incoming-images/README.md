# Incoming Images

Put your real pictures in this folder tree.

Do not edit code.
Just drop the images into the matching subfolder, then tell me which files you added.

Accepted formats:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

Recommended approach:
- Keep file names simple
- Use lowercase letters, numbers, and hyphens
- Example: `hero-main.jpg`

Folder use:
- `hero/`
  - homepage hero and delivery visuals
- `about/`
  - about section and learning studio photos
- `courses/`
  - one image per course
- `gallery/`
  - classroom, workshop, lab, showcase photos
- `partners/`
  - partner logos or partnership visuals
- `shop/`
  - product photos

What happens next:
1. You add the images here.
2. You message me that the images are ready.
3. I move them into the site image paths and wire them into the correct sections.

Current live site image slots are mapped in `SLOTS.md`.
